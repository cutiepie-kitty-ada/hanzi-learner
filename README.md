![GitHub Pages Build and Deploy](https://github.com/zehengl/hanzi-learner/workflows/GitHub%20Pages%20Build%20and%20Deploy/badge.svg?branch=master)

A Svelte app to learn Chinese characters

## Deploy

### gh-pages

    npm run build
    gh-pages -d public

### now

    npm run dev
    cd public
