# hanzi-learner

[![Build Status](https://travis-ci.org/zehengl/hanzi-learner.svg?branch=master)](https://travis-ci.org/zehengl/hanzi-learner)

A Svelte app to learn Chinese characters

## Deploy

### gh-pages

    npm run build
    gh-pages -d public

### now

    npm run dev
    cd public
    now
